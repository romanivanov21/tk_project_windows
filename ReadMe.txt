Сервер для Windows
Автор: Иванов Роман Витальевич (с)
Год: 2014 - 2015

Solution tk_project_windows.sh содержит проекты:
	- network_server_dll (библиотека для работы сервреа с сетью)
	- test_server (консольное приложение для тестирования библиотеки server_network_dll)
	- test_client (консольное приложение, емулируюущее работу клиента под Windows, для тестирования server_network_dll)
	- crypt_gost_28147-89 (библиотека функций криптографического алгоритма ГОСТ 28147-89)
	- gost_test (консольное приложение для тестирования библиотеки server_network_dll)
Общие папки:
	- shared_code (содержит общие заголовочные файлы, которые используются во многих проектах)
Для успешной сборки проекта понадобиться:
	- boost 1.57.0
	- Microsoft Visual Studio 2012

Порядрок сборки проекта:
	1) скачать и установить библиотеку boost 1.57.0
	2) создать переменные окружения: 
		- BOOSTDIR (путь до установленного boost)
		- BOOSTLINK (поуть до библиотек для линковки \stage\lib)
	3) снача следует собрать проект network_sevrer_dll
